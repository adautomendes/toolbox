# Windows Cleanup

A batch script for cleaning up temporary files and optimizing Windows system performance.

## Features

- Removes temporary files from Windows temp folders
- Clears browser cache and temporary data
- Empties recycle bin
- Removes old log files
- Optimizes disk space

## Usage

1. Clone or download this repository
2. Run `windows-cleanup.bat` with administrator privileges
3. Follow any on-screen prompts

## Requirements

- Windows OS
- Administrator access

## Warning

This script deletes files. Review the script before running to understand what will be removed.
